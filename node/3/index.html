Obiective

Obiectivele Asociatiei „Human Security Network” vizeaza derularea de proiecte ce constau in:

  • Protectia eficienta a mediului si desfasurarea de activitati de ecologizare;
  • Studii si implementari de proiecte in folosirea energiei regenerabile;
  • Campanii de ecologizare si de promovare a principiilor de viata eco;
  • Sustinerea turismului si a dezvoltarii rurale;
  • Sustinerea si organizarea cursurilor de recalificare a somerilor;
  • Masuri pentru reducerea somajului si pentru ocuparea fortei de munca;
  • Promovarea culturala si a excelentei in domenii legate de cultura, stiinta, etc;
  • Promovarea valorilor umane si profesionale in toate mediile sociale;
  • Activitati media si promovarea acestora prin diverse programe;
  • Activitati de cercetare in vederea realizarii scopului Asociatiei.
